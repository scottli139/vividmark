# Typst 离线支持任务计划

## 📋 需求概述

为 VividMark 添加 Typst 渲染支持，确保用户在 **离线/弱网环境** 下也能正常使用。

### 目标
- ✅ 完全离线渲染 Typst 代码块
- ✅ 零外部网络依赖（无需 CDN）
- ✅ 合理的包体积增量（< 15MB）
- ✅ 中文支持（可选方案）

### 依赖
```json
{
  "dependencies": {
    "@myriaddreamin/typst.ts": "^0.5.4"
  }

```

### WASM 文件
| 文件 | 版本 | 大小 | 来源 |
|------|------|------|------|
| `typst_ts_web_compiler_bg.wasm` | 0.5.4 | ~3MB | npm/jdelivr |
| `typst_ts_renderer_bg.wasm` | 0.5.4 | ~2MB | npm/jdelivr |

### 字体文件（最小集）
| 字体 | 大小 | 许可 |
|------|------|------|
| NewComputerModern-Regular.otf | ~200KB | OFL 1.1 |
| NewComputerModern-Italic.otf | ~150KB | OFL 1.1 |
| NewComputerModernMath-Regular.otf | ~400KB | OFL 1.1 |
| LibertinusSerif-Regular.otf | ~300KB | OFL 1.1 |
| **合计** | **~1MB** | - |
